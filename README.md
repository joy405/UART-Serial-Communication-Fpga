# Efficient Data Transmission via UART on FPGA

## Overview
This project implements a UART transmitter using Verilog HDL for FPGA-based systems.
The design converts 8-bit parallel data into serial data following the UART protocol.

## Features
- UART transmitter design in Verilog
- 9600 baud communication
- FSM-based architecture
- Baud rate generator
- Debounce logic for button input
- Verified using Vivado simulation

## Modules
- TOP_MODULE.v
- Transmitter.v
- Debounce_signal.v
- TOP_MODULE_testbench.v

## Tools Used
